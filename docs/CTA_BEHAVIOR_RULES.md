# CTA_BEHAVIOR_RULES.md

## Fixed Number
Phone / WhatsApp:
9937940267

## CTA: Call Now
Behavior:
- Opens phone dialer using tel:9937940267.

Placement:
- Homepage hero
- Header if available in design
- Contact page
- Footer
- Mobile sticky bar

Button style:
- Amber/yellow

## CTA: WhatsApp Enquiry
Behavior:
- Opens WhatsApp to 9937940267.
- Prefilled message:
  "Hello JS Auto Parts, I want to enquire about auto parts/accessories."

Placement:
- Homepage hero
- Product Categories
- Products / Product Enquiry
- Contact
- Footer
- Mobile sticky bar

Button style:
- Green

## CTA: Ask Availability
Behavior:
- Opens WhatsApp with category/product context.

Message example:
"Hello JS Auto Parts, I want to ask availability for [category/product]."

Placement:
- Category cards
- Product enquiry sections
- Homepage category highlights if applicable

## CTA: Request Quote
Behavior:
- Opens Request Quote page when used as navigation.
- On form submit, opens WhatsApp with formatted quote request.

Message format:
"Hello JS Auto Parts, I want to request a quote.

Name:
Phone:
Vehicle type:
Part/accessory needed:
Quantity:
Details:"

## CTA: Send Photo on WhatsApp
Behavior:
- Opens WhatsApp with message:
  "Hello JS Auto Parts, I need help identifying a part. I will send a photo here."

Important:
- Website should not upload photos in MVP.
- User attaches the photo inside WhatsApp.

## Forbidden CTA Copy
Do not use:
- Buy Now
- Add to Cart
- Checkout
- Pay Online
- Order Now
- Track Delivery
