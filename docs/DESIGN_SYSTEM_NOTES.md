# DESIGN_SYSTEM_NOTES.md

## Design System Name
JS Auto Parts — Premium Garage System

## Brand Personality
- Local trust
- Mechanical strength
- Fast enquiry
- Road-ready practicality
- Premium garage atmosphere

## Color System
Primary background:
- Carbon Black: near-black base
- Deep Graphite: section backgrounds
- Tyre Rubber: textured dark surfaces

Surface:
- Metallic panels
- Dark cards with subtle gradients
- Chrome edge highlights

Primary accent:
- Electric Blue for focus, active states, rim light, motion streaks

CTA accent:
- Amber/Yellow for Call and Request Quote
- WhatsApp Green only for WhatsApp-specific actions

Text:
- High-contrast white or near-white
- Muted steel grey for secondary text
- Avoid low-contrast grey on black

Status:
- Success green for success states
- Brake red for error states
- Amber for warnings

## Typography
Headings:
- Bold condensed sans serif
- Strong automotive feel
- Tight but readable letter spacing

Body:
- Clean readable sans serif
- Avoid overly decorative fonts

Labels:
- Small uppercase labels allowed
- Must remain legible on mobile

Buttons:
- Bold, clear, tactile
- Text labels required with icons

## Layout
Desktop:
- 12-column responsive layout
- Wide cinematic hero
- Clear CTA zone above fold
- Product/card grids with consistent gaps
- Large section spacing

Mobile:
- Single-column priority
- Bottom Call / WhatsApp bar
- Large tap targets
- Avoid dense product walls
- Hero visual must not push CTA too far down

## Spacing
Use consistent spacing scale:
- 4px micro
- 8px compact
- 12px tight
- 16px standard
- 24px section internal
- 40px large section
- 64px desktop section

## Radius
- Small controls: 8px
- Cards: 16px
- Hero panels: 24px
- Pills/badges: full rounded

## Shadows and Effects
- Dark soft shadows
- Blue rim lighting on key cards
- Amber glow near CTAs
- Avoid heavy neon everywhere
- Use texture sparingly

## Buttons
Primary call button:
- Amber/yellow background
- Dark text
- Phone icon plus text

WhatsApp button:
- WhatsApp green
- White or near-white text
- WhatsApp icon plus text

Secondary button:
- Dark metallic surface
- Chrome border
- Blue hover/focus

Disabled button:
- Muted graphite
- Low emphasis
- Still readable

## Cards
Category cards:
- Dark metallic surface
- Product/icon visual
- Category name
- Short practical text
- CTA: Ask availability

Product/enquiry cards:
- Product/category name
- Vehicle type tags
- Ask on WhatsApp
- Call Shop

Trust cards:
- 25+ years
- Local Khordha shop
- Wide product range
- Open 9:00 AM to 8:00 PM

## Forms
Quote form:
- Dark panel
- Clear white labels
- Blue focus rings
- Large inputs
- Minimal required fields
- Call/WhatsApp fallback visible

## Accessibility Rules
- Maintain strong contrast
- Text over images must have dark overlay
- Buttons must have text labels
- Icons cannot stand alone
- Mobile tap targets should be large
- Support reduced motion
- Form errors should include clear written messages
- Do not rely only on red/green for state

## Website-Specific Rules
- Keep Call and WhatsApp visible in header
- Floating WhatsApp button on desktop
- Mobile bottom Call / WhatsApp bar
- SEO-friendly headings
- Contact details repeated in footer
- No fake map unless real map integration is confirmed
- No fake reviews
